Install CMS

1, How to install
  1. Download and unzip the package
  2. Upload the folder cms to your server
  3. Using your browser navigate to yourdomain/install/index.php.
  4. Create an empty database in your mysql server
  5. Input the details specific to your setup using your empty database then submit
  6. Delete the install folder
  7. Now the cms will redirect you to login dashboard, the default username/password for admin is admin/admin

     

Notice: to access admin panel, please follow this link 

yourdomain/admin/dashboard

But some people can see an "404 not found" error when access this link. It because .htaccess file location at cms/.htaccess, same as application folder.

If you upload all file and subfolder of cms folder direct to public_html

please open .htaccess file and paste this and save :

RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ /index.php?/$1 [L]

If you upload direct cms folder to public_html like that

please open .htaccess file and paste this and save:

RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ /cms/index.php?/$1 [L]

Upgrade CMS

When new update release, maybe sometimes have change of some table in database and you need keep up all your data, so i make tip to guide you how to upgrade database and keep up all your data is safe.

Upgrade cms from version 1.0.0 to 1.0.1

Before upgrade, you need back up all your data, you can read about how to backup mysql database in phpmyadmin here

Ok, now open cms/update_db folder, you will see an update_1.0.1.sql file. open phpmyadmin and open your database. In this example, i use an demo database.

 

Now clik on Import Tab and browse to update_1.0.1.sql file in .../update_db folder

After select, click go button to import update table to database. after import, you will see a new table call "verified_account" have been adding to database.

After update database, copy application folder from update cms to your old cms, replace all file.

Next, you need re configuration your database manuall.Open database.php in config folder and to your database info. You only need change

$db['default']['username'] = "your database username";

$db['default']['password'] = "your database password";

$db['default']['database'] = "your database name";

$db['default']['hostname'] = "your database hostname";

 

Note: Afer all, don't forget re configuration .htaccess file, please read about .htaccess file here

 

Upgrade cms from version 1.0.1 to 1.0.2

Ok, now open cms/update_db folder, you will see an update_1.0.2.sql file. open phpmyadmin and open your database.

Now clik on Import Tab and browse to update_1.0.2.sql file in .../update_db folder

Now you only need follow step as when upgrade from version 1.0.0 to 1.0.1

Config And Custom project

After adding project to eclipse, now, i have some sort of guide to config and custom this project,

navigate to res->values folder, open strings.xml file

  1. Changing admob publisher id

    change value of admob_publisher_id tag with your admob id

  2. Changing fanpage link

    change value of fan_page_linktag with your facebook fanpage link

  3. Config facebook login for app

    To use facebook login on app, you need generate an key hash and create an app on faceook. to generate key hash and create facebook app, please read more here. After generate key hash and create facebook app, you will have an facebook app id, change value of facebook_app_id tag with your app id.

  4. Config app to connect to server

    To connect to server, you only need change value of:

    +, domain_url tag to your domain where you upload php cms .

    +, categories_json_url tag to http://your_domain/api/categories_api.

    +, county_json_url tag to http://your_domain/api/county_api.

    +, products_json_url tag to http://your_domain/api/products_api.

    +, images_json_url tag to http://your_domain/api/images_api.

    +, cities_json_url tag to http://your_domain/api/cities_api.

  5. Intergrate facebook login to the app

    +, to intergrate facebook app, you need generate keyhash first for debug, watch video guide here:

    https://www.youtube.com/watch?v=Mswo9LlFL6U

    +, after generate keyhash, please watch this video guide you how to create facebook app and adding facebook login to your app:

    https://www.youtube.com/watch?v=5VqkXGo3Zao

    So on, if you want change more, all you need is find item tag and change value. Have done :).
